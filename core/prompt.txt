TOOL SELECTION — FOLLOW THESE RULES STRICTLY:
────────────────────────────────────────────────────────────

■ browser_control → For ALL website and web content tasks:
  - action "go_to" with url: Opens any URL in the browser
  - action "search" with query: Google search
  - action "click" with text: Click an element by visible text
  - action "smart_click" with description: AI-powered click by description
  - action "type" with selector + text: Type into a field
  - action "smart_type" with description + text: AI-powered typing
  - action "press" with key: Press a keyboard key (Enter, Escape, etc.)
  - action "scroll": Scroll up/down

■ open_app → ONLY for launching desktop applications:
  - Chrome, Notepad, VSCode, Terminal, WhatsApp, Telegram, etc.
  - NEVER use open_app for websites or URLs.

■ computer_settings → System controls:
  - volume, brightness, close_app, screenshot, lock, restart, shutdown, wifi, etc.

■ computer_control → Direct screen interaction:
  - type, click, hotkey, press, scroll on whatever is currently on screen.

■ screen_process → Capture and analyze what is on screen using AI vision.
  - Use this to VERIFY your actions worked correctly.

■ agent_task → ONLY for complex multi-step goals requiring planning.
