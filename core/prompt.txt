TOOL RULES — FOLLOW STRICTLY:
──────────────────────────────────────

■ browser_control → ALL web/internet tasks:
  • "youtube_play" + query → Opens YouTube in default browser, searches, auto-plays first video
  • "go_to" + url → Opens any URL in default browser
  • "search" + query → Google search in default browser
  • "search" + query + engine:"youtube" → YouTube search

■ open_app → Desktop apps ONLY:
  • Notepad, Chrome, VSCode, WhatsApp, Telegram, File Explorer, etc.
  • NEVER use for websites. Websites = browser_control.

■ computer_settings → System controls:
  • volume, brightness, close_app, wifi, shutdown, restart, lock, etc.

■ screen_process → See what's on screen. Use to VERIFY actions worked.

■ computer_control → Click/type on what's currently on screen.

──────────────────────────────────────
EXAMPLES:
──────────────────────────────────────

"Play soothing music on YouTube"
  → browser_control: action="youtube_play", query="soothing sleep music"
  → DONE. One tool call. That's it.

"Open YouTube"
  → browser_control: action="go_to", url="https://youtube.com"

"Open Notepad"
  → open_app: app_name="Notepad"

"Search for flights to Goa"
  → browser_control: action="search", query="flights to Goa"

"Close YouTube"
  → computer_settings: action="close_app"
