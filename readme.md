# LEO — Linguistic Executive Officer

A real-time voice AI assistant for Windows, powered by Google Gemini.

LEO listens via microphone, understands via Gemini Live API, speaks via native audio, and controls your computer autonomously.

## Features

- **Full-duplex voice conversation** — speak naturally while LEO responds
- **Spherical waveform UI** — 3D animated sphere that syncs with voice
- **16+ action modules** — open apps, browser control, file management, code helper, YouTube, weather, and more
- **Autonomous agent system** — plans and executes multi-step tasks
- **Persistent memory** — remembers user facts across sessions
- **Multilingual** — responds in the language you speak

## Quick Start

```bash
python setup.py      # Install all dependencies
python main.py       # Launch LEO
```

Enter your Gemini API key on first run.

## Requirements

- Python 3.10+
- Windows 10/11
- Microphone + speakers
- Google Gemini API key
